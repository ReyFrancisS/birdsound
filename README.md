🐦 Philippine Birds Sound Imitation App
A comprehensive mobile application for learning and identifying Philippine bird species through their sounds and visual characteristics.
📱 About The Project
The Philippine Birds Sound Imitation App is an educational mobile application designed to help users learn about various bird species found in the Philippines. The app features authentic bird sounds, detailed information, and high-quality images of each species, making it an invaluable tool for bird enthusiasts, students, and nature lovers.
✨ Key Features

🔐 User Authentication - Secure login and signup system
🔍 Bird Species Database - Comprehensive list of Philippine birds
🎵 Authentic Bird Sounds - Listen to real bird calls and songs
📸 Visual Gallery - High-quality images of each bird species
📖 Detailed Information - Learn about habitat, behavior, and characteristics
👤 User Profile Management - Edit profile and add profile pictures
🔒 Password Security - Change password/security
📞 Support System - Contact support for assistance
ℹ️ About Section - Meet the developers and learn about the app

🚀 Getting Started
Prerequisites
Before running the application, make sure you have the following installed:

Node.js (v22.20.0)
npm or yarn
React Native CLI or Expo CLI
Android Studio (for Android) or Xcode (for iOS)

Installation

Clone the repository

bashgit clone https://github.com/ReyFrancisS/birdsound.git

Navigate to project directory

bashcd philippine-birds-app

Install dependencies

bashnpm install
# or
yarn install

Start the development server

bashnpm start
# or
yarn start

Run on your device

bash# For Android
npm run android

# For iOS
npm run ios
📖 How To Use
1. Sign Up & Login

Open the app and create a new account with your credentials
After signing up, log in using your username and password

2. Explore Bird Species

Browse through the comprehensive list of Philippine birds on the homepage
Use the search function to quickly find specific bird species

3. View Bird Details

Tap the "View" button on any bird to see detailed information
Listen to authentic bird sounds
View high-quality images
Read comprehensive information about the species

4. Profile Management

Access the menu bar and select "Profile"
View and edit your personal information
Add or change your profile picture
Update your password for security

5. Get Support

Access "Contact Support" from the menu
Connect with our team for any concerns or questions
